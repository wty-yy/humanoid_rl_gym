# G1 XML/URDF Description

- [g1_29dof.xml](./g1_29dof.xml): Used by [unitree_mujoco](https://github.com/unitreerobotics/unitree_mujoco) repository, with original 29 DoF and meshes. This file is used for **Sim2Sim testing**.
- [g1_29dof_rev_1_0.urdf](./g1_29dof_rev_1_0.urdf): Updated version with two hooks on the torso, waist roll and yaw link.
- [g1_29dof_mode_15.urdf](./g1_29dof_mode_15.urdf): Version with mode 15, which has 5010 wrist motors and lock waist (2025 G1 version). This file is used for **IsaacGym training**.
